## English

Welcome to the official repository of **Ahmed ElHediny's Portfolio Website**! This website is a premium, cutting-edge Progressive Web App (PWA) designed to showcase creative graphic design masterpieces, featuring elegant layouts, dynamic animations, and optimal media delivery.

### ✨ Key Features

*   **Premium Glassmorphism Design**: Elegant aesthetics utilizing smooth gradients, blurry glass elements, and micro-interactions that respond dynamically to user cursor movements.
*   **Progressive Web App (PWA)**: Completely installable as a native app on mobile or desktop devices. Equipped with offline caching and service workers for rapid loading speeds.
*   **Performance Optimized**: Serves modern `.webp` images dynamically with high-fidelity fallbacks using standard HTML5 `<picture>` elements. All static images are highly optimized for fast response.
*   **Dynamic Filtering & Gallery Carousel**: Fully interactive projects grid with dynamic filtering by service (Full Brand Identity, Logo Design, Social Media, Photo Manipulation, Marketing).
*   **Light / Dark Themes**: Seamless transitions between a curated bright aesthetic (Warm White & Pink) and a premium dark mode (Deep Purple & Cyan).
*   **Formspree Contact Integration**: Completely serverless contact form that securely forwards business inquiries.
*   **Git LFS Integration**: Large graphics and media files are tracked via Git LFS (Large File Storage) for a streamlined, lightweight repository size.

---

### 🛠️ Technology Stack

*   **Core Structure**: Semantic HTML5 & Modern CSS3 Custom Properties.
*   **Design Framework**: TailwindCSS (Utility-first CSS styling) combined with Custom Glassmorphism styles.
*   **Interactivity**: Vanilla JavaScript ES6 (Optimized modular events, local state management).
*   **Offline/PWA Capability**: Workbox Service Worker caching system, and fully compliant `manifest.json`.
*   **Deployment**: Optimally configured for zero-configuration Vercel Serverless static hosting.

### 🚀 Getting Started

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/ahmedelhediny32/My-Official-Website.git
    cd My-Official-Website
    ```
2.  **Run Locally**:
    Simply open the `index.html` file in any modern web browser or run it using a local server extension (e.g., Live Server in VSCode).
3.  **Development Tools** (Optional - for image processing):
    ```bash
    cd tools
    npm install
    node optimize.js   # Run the image compression utility
    ```

## العربية

مرحباً بك في المستودع الرسمي لـ **موقع معرض أعمال أحمد الهيديني**! هذا الموقع عبارة عن تطبيق ويب تقدمي (PWA) فائق الجودة والسرعة، تم تصميمه لعرض روائع التصميم الجرافيكي الإبداعي، ويتميز بتصاميم أنيقة، حركات ديناميكية، وتوزيع فائق السرعة للوسائط المتعددة.

### ✨ المميزات الرئيسية

*   **تصميم زجاجي عصري (Glassmorphism)**: واجهات بصرية فائقة الجمال تعتمد على التدرجات اللونية الناعمة، العناصر الزجاجية الضبابية، وتأثيرات التفاعل الدقيقة التي تستجيب لحركة المستخدم.
*   **تطبيق ويب تقدمي (PWA)**: قابل للتثبيت بالكامل كبرنامج مستقل على الهواتف والأجهزة المكتبية، مجهز بكاش غير متصل بالإنترنت ومشغل خدمة (Service Worker) لسرعات تحميل فائقة.
*   **أداء بصرى محسن ومضغوط**: يعرض صور `.webp` الحديثة والمضغوطة تلقائياً مع توفير صيغ بديلة (Fallback) باستخدام عناصر HTML5 `<picture>` لضمان التوافقية والأداء الفائق.
*   **نظام فلترة ومعارض ديناميكي**: لوحة مشاريع تفاعلية بالكامل تتيح تصفية الأعمال حسب التخصص (الهوية البصرية الكاملة، تصميم الشعارات، السوشيال ميديا، الدمج الرقمي، والتسويق).
*   **وضع ليلي ونهاري ناعم**: انتقال سلس وذكي للغاية بين وضع مبهج (أبيض دافئ ووردي) ووضع ليلي فاخر (بنفسجي غامق وسماوي).
*   **نموذج اتصال سحابي**: نموذج تواصل متكامل متصل بخدمة Formspree لاستقبال طلبات الأعمال مباشرة وبشكل آمن تماماً.
*   **تكامل Git LFS**: استخدام تقنية تتبع الملفات الضخمة لضمان بقاء مستودع الأكواد خفيفاً وسريعاً للغاية عند الرفع والتحميل.

---

### 🛠️ التقنيات المستخدمة

*   **الهيكل الأساسي**: لغة HTML5 الدلالية وتنسيقات CSS3 المتقدمة.
*   **إطار التصميم**: TailwindCSS مدعوماً بلمسات بصرية وخصائص مخصصة للـ Glassmorphism.
*   **التفاعلية والمنطق**: لغة JavaScript ES6 الخام لضمان أداء فائق وسلاسة في الأحداث بدون أي مكتبات ثقيلة.
*   **العمل دون اتصال (PWA)**: نظام كاش الخدمة عبر ملف `sw.js` وتهيئة PWA متكاملة بملف `manifest.json`.
*   **الاستضافة والرفع**: مهيأ بشكل مثالي للاستضافة السحابية المباشرة والخالية من الإعدادات على منصة Vercel السحابية.
